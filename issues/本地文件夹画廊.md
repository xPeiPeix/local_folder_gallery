# 本地文件夹画廊项目

**开始时间：** 2025-01-27 13:56:16

## 项目概要
制作一个本地文件夹的画廊展示系统，支持递归搜索图片文件，展示图片信息，提供放大查看和批量导出功能。

## 核心功能
- [x] 文件夹选择与递归扫描
- [x] 图片信息解析与展示
- [x] 图片放大查看器
- [x] 批量选择与导出功能
- [x] 美观简洁的界面设计

## 技术栈
- 前端：HTML5 + CSS3 + JavaScript
- 文件处理：File API + FileReader
- 图片处理：Canvas API
- 导出功能：JSZip 库
